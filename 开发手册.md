# 开发手册
## 创建项目
```shell
git clone git@github.com:gemiman/daoyi-vue-rs.git
```
## 初始化项目
```shell
cargo init
```
## 设置 server.port
- DY__SERVER__PORT=8080
- 完整示例
```shell
DY__SERVER__PORT=8080 cargo run --bin daoyi-demo
```
## 运行项目
```shell
ACTIVE_PROFILE=local
```
## 安装 sea-orm-cli
```shell
cargo install sea-orm-cli
```
- 生成 demo entity
```shell
sea-orm-cli generate entity -s demo --with-serde both --model-extra-attributes 'serde(rename_all="camelCase")' --date-time-crate chrono -o ./crates/libs/daoyi-entity-demo/src/demo_entity
```

## 输出sql日志，配置环境变量：

```text
RUST_LOG=DEBUG
```
